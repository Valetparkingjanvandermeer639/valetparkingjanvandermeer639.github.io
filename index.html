🎥 Dxtory - Record high quality game footage easily

Download Dxtory for Windows

📋 Overview

Dxtory records video from games. It captures frames from the graphics memory to provide lossless output. This software works on Windows 10 and 11. It assists players who need to save gameplay for editing or sharing.

🚀 System Requirements

Your computer needs specific hardware to run this software well. Please ensure your machine meets these marks:

  • Operating System: Windows 10 or Windows 11.
  • Processor: Intel Core i5 or AMD equivalent.
  • Memory: 8 GB RAM or higher.
  • Graphics: DirectX 9, 10, 11, or 12 compatible hardware.
  • Storage: Sufficient disk space for large video files.

⚙️ Installation Steps

Follow these steps to set up the recorder on your machine.

  1. Visit the official release page to see available versions.
  2. Select the latest version link to download the installer file to your computer.
  3. Locate the file in your Downloads folder once the transfer finishes.
  4. Double-click the installer icon to start the setup process.
  5. Follow the prompts on the screen to choose your installation directory.
  6. Click the Install button to copy the files.
  7. Select Finish when the process completes.

🎮 Configuring Your Settings

Once you open the software, go to the settings tab. You will see several options for video and audio. The software captures data directly from the graphics card. This means it creates high-quality files. You should choose a fast storage drive to save your recordings. Slow hard drives can cause stutters during the capture process.

Check the hotkeys tab to set your start and stop recording keys. Pick keys that do not interfere with your game controls. Test these settings in a short session before you start a long recording.

🛡️ Performance Tips

Recording gameplay uses machine resources. If you notice lower frame rates in your game, try these adjustments:

  • Reduce the recording frame rate in the software settings.
  • Check that your storage drive has plenty of free space.
  • Close background applications to free up memory.
  • Use a separate physical drive to store recording files if possible.

💡 Frequently Asked Questions

Does this software capture video from any game? It works with most DirectX and OpenGL games. Some titles have restrictions that prevent recording.

Where do my files go? You can set the save folder in the main settings tab. Check this folder after you finish a recording session.

Can I record the microphone audio? Yes. Go to the audio tab and select your recording device from the list.

Maintain your capture settings for optimal results. Ensure your drivers stay updated for the best performance.
